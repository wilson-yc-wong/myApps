Overview

Here is collection of my works. They are technical interview tests, and implementation of concepts in my brain.

Technical design

All applications are mainly developed by Java, publish to GitHub, and run on AWS.
They also can run on local machine by simple maven spring-boot command "mvn spring-boot:run". If you have installed Java SDK 8 and Maven.
Following technologies have been used
  • springboot
  • angularjs
  • spring-integration
Following non-functional requirements are considered
  • Authorization
  • Authentication
  • Testability
  • Integrability
  • configuration management
  • transaction management
  • support management
